# Axis 1 · App Icon 書法飛白 v1

## 方向

第一版 app icon 探索。核心概念是一道書法快筆，速度快到收筆裂出飛白，也就是筆毛分岔的白絲。飛白是速度的物理證據，只有夠快的一筆才有。橘藏在飛白裡一絲，延續品牌橘色克制。

## 變體

| 筆勢 | 動勢 | 說明 |
|---|---|---|
| 掠 Glide | 橫向掠過 | 手腕一抖，左到右橫掃，最接近 swish 聲 |
| 捺 Stroke | 斜向落定 | 書法的捺，起輕行沉收筆出鋒 |
| 繞 Loop | 一筆迴轉 | 一圈繞成，開口不封閉，避正圓 |
| 頓 Drop | 縱向落下 | 像記一個數字的那一豎，頂穩收尖 |
| 疊 Weave | 雙筆交織 | 兩筆交會不平行，避雙速線 |
