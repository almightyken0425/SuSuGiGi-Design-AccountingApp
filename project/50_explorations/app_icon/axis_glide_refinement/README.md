# Axis 6 · App Icon 深化場（Glide + K15）

## 方向

使用者綜覽 Axis 1 至 5 後圈定兩條深化軌、集中在本 axis：

- **Glide 軌**：v1 書法飛白的掠 Glide 翻案深化、Axis 1 的 Rejected 狀態部分翻案。參數化 Lab、主體與飛白繪法自 `swish_engine.jsx` 移植成可掃參數版、不動共用 engine。
- **K15 軌**：Axis 5 因子表的 K15（多條 × 深淺 × 直棒 × 無暈 × 斜升）出八個深化變體。

深化依三步走：先驗生死關、再掃方向與參數、最後手工精修出定稿候選。

## 內容結構

| family | 內容 | 目的 |
|---|---|---|
| 生死關 | Glide 尺寸階梯 180 → 44、深淺桌布 60px 情境 | 飛白絲在小尺寸的存活驗證 |
| 方向軸 | Glide 0° / -12° / -20° | 水平對斜升的 energy 比較 |
| 小因子表 | Glide tailStart 裂點 × 飛白密度 × 三種底 | 單軸效應並排目測 |
| K15 深化 | 基準、粗細階、前緣切齊、條內漸層、平切端、四條、深紫底、極光底 | 直棒三條的變體比較、各附 120 與 44 兩級 |
| K15-4 再深化 | 尾透程度、有無條間階、粗細階、錐形條、平切端、深底反白、橘火花 | 條內漸層軌收斂、各附 120 與 44 兩級 |
| D3 再深化 | 粗細階有無、尾透、錐形、微弧、平切端、深底、極光加橘 | 純條內漸層軌收斂、各附 120 與 44 兩級 |
| E5 再深化 | 弧度掃描、粗細階、微弧錐形、尾全透、深底、極光加橘 | 微弧×純條內漸層軌收斂、各附 120 與 44 兩級 |
| 角度掃描 | E5 基底、走勢角 0 到 45 六檔 | 走勢角效應並排目測、fade 隨整組旋轉 |

Glide 可掃參數：`angle`、`tailStart`、`density`、`bg`、`seed`。底色 deep 時自動換淺筆、飛白 destination-out 露深底。

## 與其他 axis 的關係

- Axis 1 書法飛白：本 axis 的翻案來源、幾何沿用其 glide cfg。
- Axis 3 六形態：非書法 Glide 仍在該處、兩線未合流。
- Axis 5 因子實驗：斜升優於水平、底色三選的結論由此帶入。
