# Axis 2 · App Icon 非書法速度 v2

## 方向

第二版。書法否決後改走抽象速度：讓一個紫色的存在在加速中改變形態，被拉長、流動、拖尾、暈開。不靠符號、不靠箭頭，形態本身就是速度。橘只一絲。

## 變體

| 形態 | 動勢 | 說明 |
|---|---|---|
| 速度變形 Morph | 一個形被甩長 | 飽滿流線體，圓頭加尖端，立體高光 |
| 流線流體 Flow | 一縷絲滑流動 | 帶絲綢漸層，橘一縷織在其中 |
| 拖尾動勢 Comet | 凝聚頭加漸隱尾 | 頭實尾虛化，連續非散落 |
| 漸層光暈 Glow | 一團流動的光 | 無硬邊，純漸層，最抽象 |
