# Axis 3 · App Icon 六形態 v3

## 方向

第三版收斂。保留 v2 的速度變形 Morph 與 v1 的橫掠 Glide 當底，再新增四個變化。聚焦一個主體加速度的形態家族。六個橘份量皆 0.14 至 0.22%，極小均衡。

## 變體

| 形態 | 來源 | 動勢 |
|---|---|---|
| 速度變形 Morph | 保留 v2 | 一個形被甩長 |
| 橫掠 Glide | 保留 v1 書法 | 左到右橫掃 |
| 拉絲 Draw | 新 | 頭凝聚，尾拉絲 |
| 輕掠 Wisp | 新 | 輕順橫掠，末端化開 |
| 疊影 Echo | 新 | 主體加前方淡影 |
| 弧掠 Arc | 新 | 橫向實心弧帶 |

## 共同約束

紫 `#4323A0` 主體、橘 `#F24F13` 一絲、底 `#F2F2F7`。速度是主角、橘只一絲、自然不對稱。明確排除：記號、字母、箭頭、太極、正圓、尖角、雙速線、散落墨點、書法。
