# Axis 5 · App Icon 因子實驗 K / L / N

## 方向

把 Axis 4 使用者圈選歸納的四元素化成二元軸、正交展開成因子表。每格只動軸值、其餘全固定，讓單一元素的效應可並排目測。

## 軸與區塊

| 區塊 | 軸 | 格數 |
|---|---|---|
| K | 條數（一/多）× 深淺（無/有）× 線形（彎/直）× 光暈（無/有），斜升 | 16 × 深白兩底 |
| L | 同 K 四軸，方向改左到右 | 16 × 深白兩底 |
| N | 形定 K13 與 K5，材質（平塗/玻璃）× 底型（深紫平/深紫漸層/白平/極光漸層） | 16 |

玻璃＝GlassView 配方擬態：柔影、半透明體、上緣白高光。底型的漸層取自品牌 hero 漸層與 Axis 2 極光背景。

## 已知結論

- 彎線與深淺兩軸效果穩定正面。
- 直棒在水平向全滅：單條讀成減號、多條讀成漢堡選單。
- 光暈在單條上是流光、在多條上會糊；白底上光暈變墨暈。
- 玻璃在深底與極光底成立、在白底近隱形。
- 深紫漸層的亮角會吃掉深淺階的淡條。

## 候選

K13 多條彎深淺無暈、K6 單條流光、L13 水平風紋、N1 即 K13 深紫底、N2 深紫玻璃、N13 極光底平塗。canvas 上以橘點標記。
