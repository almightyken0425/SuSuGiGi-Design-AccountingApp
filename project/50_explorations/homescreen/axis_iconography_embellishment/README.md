# Axis 3 · Iconography & Embellishment

從 icon 風格著手探索 HomeScreen 的視覺風格。配色維持 Classic Purple、質感維持 Liquid Glass（與 Axis 2 current direction 呼應）。每個 variant 的 icon 主題貫穿到 Header / Period switcher / Focus row / Section header / 單筆紀錄 recurring / FAB 等所有 icon 出現的位置。

## 並陳的提案

| Variant | Icon 形態 | 性格 |
|---|---|---|
| V1 · Glassmorphic Chip       | 玻璃方形 chip + 主色 icon       | 最 native iOS、最輕 |
| V2 · Monochrome Filled       | 實心主色 chip + 白色 icon       | 對比最強 |
| V3 · Soft Pastel Tile        | 淺主色 tile + 主色 icon         | 柔和、最不搶眼 |
| V4 · Glass Sphere            | 球形玻璃 + radial highlight     | 立體感、最有觸感 |
| V5 · Soft Embossed           | 淺色 + inset shadow 凹陷感      | 觸控按鍵、neumorphism 輕版 |
| V6 · Sticker                 | 白底 + 粗主色描邊 + offset 陰影 | 童趣、tactile、貼紙感 |
| V7 · Tag Cut Corner          | 主色實心 + 右下斜切（票根感）   | 特殊形狀、最有性格 |
