# Axis 4 · Personality (packaged)

把整體性格當作一次性定位的完整 packaged 主題——橫跨色、質感、字體、icon、裝飾。

**規則：**
- 配色：每個 variant 主色系（深淺）+ 至多 1 強調色
- 質感：以 Liquid Glass 為主軸（部分 variant 走特殊質感）
