# Axis · Press Feedback

## 問題

- Key Visual 軸定案鍵面留用 V0 玻璃磚
- 現況按壓回饋是整鍵透明度降 0.7
- 玻璃磚本身半透明
    - 透明度變化在磚面上讀感弱
    - 快速連點時幾乎看不到回饋
- 目標改為按下變色
- 鍵面、排列、底座維持 V0 不動
- 全部 artboard 可互動
    - 按住任一鍵看回饋
    - 放開看 160ms 回復動畫

---

## 並陳的提案

| Variant | 變色位置 | 數字鍵按下 | operator 鍵按下 | 性格 |
|---|---|---|---|---|
| P0 現況 · Opacity dim | 整鍵透明度 | 透明度 0.7 | 透明度 0.7 | 最保守、讀感弱 |
| P1 · Purple tint | 磚面底色 | 染 p50 | 染 p100 實色 | 品牌色輕染 |
| P2 · Solid flash | 磚面加文字 | 轉 p500 白字 | 轉 p500 白字 | 對比最強 |
| P3 · Ink dim | 磚面底色 | 淡墨蓋色 | p200 半透加深 | iOS 鍵盤感 |
| P4 · Ring highlight | 內圈框 | p300 內框亮起 | p300 內框亮起 | 磚面不動 |
| P5 · Scale + tint | 磚面加尺寸 | 染 p50 縮 0.94 | 染 p100 縮 0.94 | 下沉觸感 |

---

## 各 variant 要點

- **P0 現況:**
    - baseline、供比對用
- **P1 · Purple tint:**
    - 與品牌色系一致、不搶金額顯示
    - 數字與 operator 維持深淺區分
- **P2 · Solid flash:**
    - 回饋最明確、誤觸最易察覺
    - 快速連點時畫面閃動感最強
- **P3 · Ink dim:**
    - 中性不帶品牌色
    - 與系統鍵盤行為直覺一致
- **P4 · Ring highlight:**
    - 磚面材質完全不動
    - 小鍵上內框存在感偏弱
- **P5 · Scale + tint:**
    - 觸感回饋最擬物
    - 縮放與相鄰鍵視覺間距會變動
